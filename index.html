Question: How does Javascript works?


Answer: Javascript is an interpreted language that works instantly as soon as the command is given. The engine that runs Javascript in the browser is the V8 engine. When Javascript code is loaded, The V8 engine executed the code from top to bottom. This interprets the codes and converts the codes into machine language. But to make Javascript work faster it also uses the V8 JIT (Just in time) compiler, which optimizes the codes.



Question: What is the difference between local storage and session storage?


Answer: Data is stored permanently in Local Storage. But the data in the Session Storage is temporary. Local Storage data won't expire if the browser is closed or open the other tab. But Session Storage data will clear when the browser is closed. Local storage is not session based, so we have to delete local storage data with javascript or manually.Session storage is session-based, so data is stored only for the duration of the browser or tab is open. The data will autometically deleted with the browser or tab is closed.
